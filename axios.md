# axios
axios是一个基于promise封装好的请求库，作用于node.js 和浏览器中，nodejs采用的是http模块，浏览器采用xmlHttpRequest请求。

特性：
1、从浏览器创建 XMLHttpRequests
2、从 node.js 创建 http 请求
3、支持 Promise API
4、可以添加拦截和响应的拦截器
5、可以转换请求和响应数据
6、发出去的可以取消请求
7、自动转换JSON数据
8、客户端支持防御XSRF

# 为什么使用axios
使用axios的目的，一个是axios支持promse api风格的请求，axios是可以通过配置进行统一下的控制
